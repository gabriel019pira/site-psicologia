## 📝 Próximas Ações - Ordenadas por Prioridade

### 🔴 CRÍTICO (Próximos 3 dias)

1. **Preencher dados reais do Schema.org**
   - Abra: index.html
   - Encontre a seção `<script type="application/ld+json">`
   - Substitua [VALORES ENTRE COLCHETES]:
     - Endereço completo
     - Telefone com +55
     - Email
     - Links de redes sociais

2. **Registrar no Google My Business** ⭐ Mais importante para SEO Local
   - Acesse: https://www.google.com/business
   - Crie negócio: "Dra. Franciele Silva - Psicóloga"
   - Selecione categoria: "Psicólogo"
   - Complete todos os campos
   - Tempo: 15 minutos = máximo impacto

3. **Registrar no Google Search Console**
   - Acesse: https://search.google.com/search-console
   - Use meta tag HTML para verificação
   - Cole meta tag em: `<head>` do index.html
   - Envie sitemap.xml

4. **Atualizar Canonical URL**
   - Procure: `<link rel="canonical"`
   - Altere para seu domínio real
